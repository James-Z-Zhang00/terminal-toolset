# operating-system-toolset

My system setup tools, configuration and personal developed mini programs to help improving productivity, including MacOS and Windows platforms

### List
- `1-zsh_alias.sh` - some command shortcuts, including basic directory change, opening apps and folders
- `2-folder-descrition.sh` - allows to write a description for rarely used folders, the description will display in terminal when switching to the directory. The description can be removed by `rm .folder_description`
- `3-selective-cross-device-file-sync` - device agnostic file synchronization system for selected files, work with Google Drive
